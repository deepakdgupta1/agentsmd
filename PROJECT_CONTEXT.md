# AGENTS.md Generator - Project Context

## Vision
A platform to help users (humans or AI agents) create optimized AGENTS.md configuration files for AI agents in different roles.

## What is AGENTS.md?
A configuration/instruction file that defines:
- Agent behavior and capabilities
- Role-specific instructions
- Context and constraints for the AI agent

## Example Use Cases
- Software Tester agent
- Software Architect agent
- Digital Marketing agent
- Time Management Coach agent

## User Interaction Modes (All Supported)
1. **Interactive Questionnaire** - Guided questions to define the agent
2. **Template Library Browser** - Pre-built templates for common roles
3. **Natural Language Description** - Describe role, AI generates AGENTS.md

## Content Strategy (Hybrid Approach)
- **Curated Templates** - Pre-written templates for various roles
- **AI Generation** - LLM generates custom content on-the-fly
- **Community Contributions** - Users share and rate templates
